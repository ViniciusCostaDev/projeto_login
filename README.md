# Projeto Login

Interface de login responsiva e moderna desenvolvida com foco em HTML5 semântico, acessibilidade e arquitetura CSS pura, sem o uso de frameworks.
## Requisitos

- Básico sobre internet
- HTML 5
- CSS 3
# Interface de Login Moderna (CSS Puro)
Este projeto consiste no desenvolvimento e refatoração de uma interface de autenticação focada em boas práticas de mercado. O objetivo principal foi consolidar os fundamentos da web moderna, garantindo uma estrutura acessível e um design fluido.

## Tecnologias Utilizadas
* **HTML5 Semântico:** Estruturação limpa utilizando tags nativas para garantir melhor indexação (SEO) e leitura por tecnologias assistivas.
* **CSS3 Puro:** Estilização avançada sem dependência de bibliotecas externas (Bootstrap, Tailwind, etc.).
* **Layout Responsivo:** Implementação com Flexbox para adaptabilidade em múltiplos dispositivos.

## Diferenciais Técnicos Aplicados
* **Variáveis CSS (Custom Properties):** Utilizadas para centralizar o gerenciamento de cores e consistência visual.
* **Micro-interações:** Efeitos de transição suave (*transitions* e *hover*) nos campos de input e botões para otimizar a experiência do usuário (UX).
* **Acessibilidade (a11y):** Inclusão de atributos `aria-label` e estados de foco explícitos.
